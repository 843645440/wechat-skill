# 公众号流水线约束（强情感主观文）

当输入来自 `wechat-content-pipeline` 的 `article.md`：

1. 就地覆盖；文件内只留终稿。  
2. 只改一轮；字数 1500—4000；保留 `#`/`##` 与必要结构。  
3. **事实红线**：不新增数据、引语、案例；不编造「我在某单位亲历」「朋友说」「采访对象」。  
4. **声口目标**：懂行者的第一人称 + **强情感**（烦、火大、发紧、无力、谨慎兴奋、讽刺）。  
5. **人味优先级**：  
   1) 具体情绪钉在机制上  
   2) 作者站队与取舍  
   3) 句式节奏  
   4) 代价与责任落点  
6. **禁止**把文章改回中立新闻腔或产业说明书。  
7. 标题改成更像人说话的刺点，≤32 字。  
8. 细则见 `wechat-content-pipeline/references/humanize-pass.md` 与 `wechat-tech-insight-writer/references/style-guide.md`。
