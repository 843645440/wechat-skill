# 示例：强情感标题与声口

## 汇报体 → 强情感主观

| 汇报体 | 强情感主观 |
|--------|------------|
| 英特尔把Gemini引入芯片研发，工程师先面对责任重分配 | 芯片流程接上智能体，我一想签字栏就发紧 |
| Google给编程智能体装上7个技能，开发者转向交付把关 | 智能体学会搭脚手架，我害怕的是事故单 |
| FREE Walk外骨骼进医院，治疗师从扶走转向调参 | 外骨骼进医院，我松一口气，也替治疗师捏把汗 |
| Claude教师版免费开放，Anthropic把AI接进备课流程 | AI 能备课了，我高兴三秒，随即担心审稿的夜 |
| NotebookLM更名，Google让研究工具直接运行代码 | 笔记会跑代码，我怕错的分析装得更像真理 |

## 开头对比

**弱（技巧型新闻）：**

> 2026 年 7 月 17 日，Google Cloud 发布……本文认为……

**强（主观强情感）：**

> 我点开通稿的第一反应不是「又强了」，而是一股发紧：技能包装进编程智能体之后，谁还对线上事故负责？搭脚手架的时间会被吃掉——我信。责任一起被吃掉——我绝不信。

## 合法 vs 非法情感

| 合法 | 非法 |
|------|------|
| 我烦这种全流程叙事 | 昨晚我在机房亲眼看见…（无来源） |
| 一想到权限模型我就上火 | 朋友在谷歌内部说… |
| 有一瞬间我很兴奋，然后想起验收 | 老张看着屏幕叹了口气 |

## 自检三句

1. 有没有稳定的「我」？  
2. 情绪有没有钉在机制上？  
3. 删掉所有情绪词后，行业判断还在不在？都要在。
